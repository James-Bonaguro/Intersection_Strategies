## How to Generate the One-Page PDF

1. Open `print.html` in Chrome
2. Press **Print**
3. Destination: **Save as PDF**
4. Paper size: **Letter**
5. Scale: **100%**
6. (Optional) Enable **Background graphics**

This will produce a clean, single-page PDF suitable for email or in-person use.
